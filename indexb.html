Project Hermes

Features

Reveal the libraries within your network

Quickly find libraries used by developers you trust.

Get recommendations while searching

Looking for a new library? Need to phone-a-friend? This is one step ahead of you. As you type, it shows the most relevant packages ordered by the number of people in your network using it. Stop learning to use libraries no one can help you with.

Add developers you trust

Invite your co-workers and friends straight from your coding environment. Now you can see what libraries they’re using in their code. When you get stuck, you’ll be able to contact friends & colleagues that you trust from your network. The more the merrier.

Connect to developers that use the same libraries

When you import a library into your code, you’ll see who's already using it within your network or organization. Now you can ask specific questions about your use-case. Chances are they can help you quickly since they’ve already implemented it. Reduce time-waste.
