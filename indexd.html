Project Hermes

Features

Stack Overflow helps...but you have to wait a while

As you're coding reach out to developers who've used the same code as you. Get un-stuck faster.

Email or slack them — but don’t be stuck while coding

When you import a library into your code, you’ll see who's already using it within your network or organization. Now you can get answers specific to your use-case.

Decide what to use, based on your coworkers & friends

Looking for a new library? Need to phone-a-friend? Hermes is one step ahead of you. As you type it shows you the most relevant and popular libraries used by your peers.

The code editor is a very lonely place

Connect with developers you trust. Invite your coworkers and friends straight from your coding environment. Now you can see what libraries they’re using in their code. The more the merrier.
