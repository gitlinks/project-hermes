Connecting People Through Code

Reach out to developers who have done it before... as you're coding.

Add developers you trust

Invite your co-workers and friends straight from your coding environment. Now you can see what libraries they’re using in their code. When you get stuck, you’ll be able to contact people in your network that you know and trust. The more the merrier.

Connect to developers that use the same libraries

When you import a library into your code, you’ll see who's already using it. Now you can ask specific questions about your use-case. Chances are they can help you quickly since they’ve already implemented it. Reduce time-waste.

Get recommendations while searching

Looking for new library? Need to phone-a-friend? This is one step ahead of you. As you type, it shows you how many and who’s already using relevant packages.
